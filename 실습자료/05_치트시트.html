🚀 Claude Code 치트시트

책상에 붙여두고 쓰세요 · 인쇄: Ctrl/⌘ + P

설치 한 번만 하면 끝

# 엔진/도구 확인
node --version
git --version
# Claude Code 설치
npm install -g @anthropic-ai/claude-code
# 실행 (다음부터 이 한 줄)
claude

명령어 이 3개만

  • /help — 명령 목록 보기
  • /clear — 대화 초기화(새 작업)
  • /exit — 종료
나머지는 한국어로 그냥 말하기

프롬프트 3원칙

  • 구체성 — "정리해줘"❌ → "월별 합계 표로"✅
  • 맥락 — "팀장 보고용, 존댓말로"
  • 단계 — "조회 → 요약 → .md 저장" 순서로

Dooray MCP 연동

  • ① Dooray 설정 → API 토큰 발급
  • ② .env 에 DOORAY_TOKEN=...
  • ③ "Dooray 내 프로젝트 목록 보여줘"
⚠️ 토큰은 비밀번호! 노출 금지

자주 쓰는 요청

# 파일 확인
이 폴더에 어떤 파일 있는지 알려줘
# 주간보고
Dooray 이번주 내 태스크로
주간보고.md 만들어줘
# 대시보드
이 CSV로 차트 대시보드 html 만들어줘

막힐 때

  • 에러 → 방금 에러 났어. 원인과 해결법 알려줘
  • 안 열림 → 터미널의 URL 복사해 브라우저에
  • 경고(warn) → 대부분 무시 OK
  • 5분 넘게 막히면 손 들기 🙋
