# A/B 실습 워크시트 (기초 과정)

> 사전설문(da.gd/HaExGw) 결과로 배정된 **본인 그룹**의 워크시트만 진행하세요.
> 각 STEP은 **① 복붙 프롬프트 → ② 이렇게 나오면 성공 → ✋ 체크포인트** 순서입니다.

---

## 🅰️ A그룹 — Dooray 자동화 (산출물: `주간보고.md`)

### 시작 전 점검
- [ ] `claude --version` 이 버전을 출력한다
- [ ] `Dooray에서 내 프로젝트 목록 보여줘` → 목록이 나온다

### STEP 1 · CLAUDE.md 작성 (10분)
템플릿(`03_CLAUDE.md_템플릿.md`)을 복사해 작업 폴더에 `CLAUDE.md` 로 저장하고, 본인 팀·보고 형식·규칙을 채웁니다.
- ✋ 체크: 폴더에 `CLAUDE.md` 가 있고 내 업무가 3줄 이상 적혀 있다.

### STEP 2 · 내 태스크 조회
```
Dooray에서 이번 주 내 태스크를 조회해서 상태별로 보여줘
```
- ② 성공: 완료/진행중 등 태스크가 목록으로 나온다.

### STEP 3 · 주간보고 생성 (3원칙 적용)
```
위 태스크로 주간보고를 만들어줘.
- 완료 / 진행중 / 다음주 계획 3개 섹션
- 팀장 보고용, 존댓말, 각 항목 한 줄 요약
- 수치는 굵게
```
- ② 성공: 3개 섹션으로 정리된 보고 초안이 나온다.

### STEP 4 · 파일로 저장
```
이 내용을 주간보고.md 파일로 저장해줘
```
- ✋ 체크포인트(완성): 폴더에 `주간보고.md` 가 생겼고 열어서 확인된다.

> 🆘 5분 넘게 막히면 손 들기. 에러는 `방금 에러 났어. 원인과 해결 방법 알려줘`.

## 🅱️ B그룹 — 대시보드 만들기 (산출물: `대시보드.html`)

### 시작 전 점검
- [ ] `claude --version` 이 버전을 출력한다
- [ ] 실습용 엑셀/CSV 파일을 받아 작업 폴더에 두었다

### STEP 1 · CLAUDE.md 작성 (10분)
템플릿을 복사해 `CLAUDE.md` 저장 후, 내 데이터·보고 목적·보고 싶은 지표를 채웁니다.
- ✋ 체크: 폴더에 `CLAUDE.md` 와 데이터 파일이 함께 있다.

### STEP 2 · 데이터 분석
```
이 폴더의 CSV 파일을 열어서 열별 요약과 이상치를 한국어로 알려줘
```
- ② 성공: 열 이름·합계·평균 등 요약이 나온다.

### STEP 3 · 대시보드 생성 (3원칙 적용)
```
이 데이터로 HTML 대시보드를 만들어줘.
- 상단에 핵심 지표 카드 3~4개
- 월별 추세는 꺾은선, 항목별 비교는 막대 차트
- 한 파일(html)로, 브라우저에서 바로 열리게
```
- ② 성공: `대시보드.html` 같은 파일이 생성된다.

### STEP 4 · 브라우저에서 확인
생성된 `.html` 파일을 더블클릭해서 엽니다.
- ✋ 체크포인트(완성): 브라우저에 차트가 그려진 대시보드가 보인다.

> 🆘 차트가 안 보이면: `대시보드가 빈 화면이야. 원인 찾아서 고쳐줘` 라고 요청.
