# 설치 가이드 — Node · Git · Claude Code · Skills · Dooray MCP

> 입문 과정(2시간)에서 함께 진행합니다. 혼자 미리 하실 때도 이 순서를 그대로 따라오세요.
> **막히면 오래 붙들지 말고 손을 드세요(조교 호출).** 컴퓨터는 명령 몇 줄로 망가지지 않습니다.

## 0. 준비물
- Claude **유료 계정** (Pro / Team / Max 중 하나)
- 노트북 (Windows 또는 Mac)
- 인터넷 연결

---

## STEP 0 · 터미널 열기
- **Windows:** 시작 → `PowerShell` 검색 → 실행
- **Mac:** `⌘ + Space` → `터미널` 입력 → 실행

검은 창이 뜨면 성공입니다.

## STEP 1 · Node.js 확인/설치 (엔진)
```bash
node --version
```
- `v20`, `v22` 같은 숫자가 나오면 → 이미 설치됨, 다음 단계로.
- 아무것도 안 나오거나 오류면 → https://nodejs.org 에서 **LTS** 버전 설치 → **터미널 재시작** 후 다시 확인.

## STEP 2 · Git 확인/설치 (파일 관리·다운로드)
```bash
git --version
```
- 버전이 나오면 OK.
- 안 나오면 → https://git-scm.com 에서 설치 → 터미널 재시작.

## STEP 3 · Claude Code 설치 (자동차)
```bash
npm install -g @anthropic-ai/claude-code
```
- 1~2분 소요. 노란 경고(`warn`)는 대부분 무시해도 됩니다.
- 마지막에 `added ... packages` 가 뜨면 성공.

## STEP 4 · 로그인 & 첫 실행
```bash
claude
```
- 브라우저가 자동으로 열립니다 → Claude 유료 계정으로 로그인.
- 브라우저가 안 열리면 → 터미널에 표시된 **URL을 복사**해 브라우저에 붙여넣기.

## STEP 5 · 첫 대화 (설치 확인)
Claude Code가 켜진 상태에서:
```
지금 이 폴더에 어떤 파일이 있는지 한국어로 알려줘
```
한국어로 답하면 **설치 성공** 🎉 → 조교에게 확인받으세요.

---

## STEP 6 · Skills 설치 (확장팩)
Skill은 Claude Code에 기능을 더하는 확장팩입니다(예: oh-my-skills).
- 강사가 안내하는 **설치 명령을 그대로 복사 → 실행**.
- 확인:
```
사용 가능한 skill 목록 보여줘
```

## STEP 7 · Dooray MCP 연동
MCP = Claude를 외부 도구(Dooray)에 연결하는 표준 케이블.

1. **토큰 발급** — Dooray → 설정 → API/개인 인증 토큰 발급
2. **.env 저장** — 프로젝트 폴더의 `.env` 파일에 붙여넣기
   ```
   DOORAY_TOKEN=발급받은_토큰
   ```
3. **연동 확인**
   ```
   Dooray에서 내 프로젝트 목록 보여줘
   ```
   목록이 나오면 연동 성공 🎉

> ⚠️ **토큰은 비밀번호입니다.** 화면 공유·채팅·캡처에 노출 금지, `.env` 밖으로 복사하지 마세요.

## 기본 명령어 3개
| 명령 | 하는 일 |
|---|---|
| `/help` | 사용 가능한 명령 목록 |
| `/clear` | 대화 맥락 초기화(새 작업 시작) |
| `/exit` | Claude Code 종료 |

나머지는 그냥 **한국어로 말하면** 됩니다.
