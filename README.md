# 클로드 코드 사내 교육 — 강의·실습 자료 (v3)

비개발 직군을 위한 Claude Code 교육. **입문(설치) → 기초(업무 자동화)** 2단계.
초점은 **회사 업무 자동화**이며, 부업/개인 프로젝트는 "이런 것도 가능하다"는 예시로만 다룹니다.

## 강의 슬라이드 (HTML · 스크린 공유/2모니터용)
- **[입문] 초보자를 위한 클로드 코드 설치 가이드** — [`입문/index.html`](./입문/index.html) · 2시간 / 50명
- **[기초] 클로드 코드로 시작하는 업무 자동화** — [`기초/index.html`](./기초/index.html) · 3시간 / 30명

### 슬라이드 조작
| 키 | 동작 |
|---|---|
| `←` `→` | 이전 / 다음 |
| `F` | 전체화면 |
| `O` | 전체 슬라이드 보기 |
| `P` | 인쇄 / PDF |
| 우상단 배지 클릭 | 번호로 점프 |

**2모니터 강의(강사 리모컨):** 슬라이드 URL 뒤에 `?remote=1` 을 붙여 휴대폰/보조 화면에서 열면
키보드 없이 넘길 수 있고, "빔 화면 새 창으로 열기"로 관객용 화면을 분리할 수 있습니다.
