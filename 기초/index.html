📖 실습자료 2-B

폴더에 이 파일을 두면 Claude가 매번 자동으로 읽고 내 맥락에 맞게 일합니다.

# CLAUDE.md 예시 ## 나는 누구 - 마케팅팀. 주간보고를 매주 금요일 작성. ## 자주 하는 일 - Dooray 태스크를 모아 주간보고(.md)로 정리 - 보고 형식: 완료 / 진행중 / 다음주 계획 ## 규칙 - 존댓말, 불릿 3개 이내, 수치는 굵게
📍 어디에 두나
작업하는 폴더 맨 위에 CLAUDE.md 파일로
🛠️ 만드는 법
/init 입력 또는 "CLAUDE.md 만들어줘"
🔁 계속 다듬기
어색한 결과가 나오면 규칙 한 줄씩 추가
💡 잘 쓰는 팁
누구·자주 하는 일·원하는 형식·규칙 4가지만 적어도 충분. 길게 쓰기보다 구체적으로.
한 번 적어두면 "내가 누구고 뭘 원하는지"를 다시 설명할 필요가 없습니다.

design.md로 사내 템플릿 맞춘 문서 만들기

📖 실습자료 3-I

Claude Code는 PPT(python-pptx)도 만들 수 있습니다. 핵심은 design.md에 사내 템플릿 규칙을 적어두는 것.

📌 design.md = 디자인 설명서
폰트·색상·레이아웃 규칙을 적어두면, Claude가 매번 같은 스타일로 PPT를 만듭니다.
# design.md 예시 ## PPT 규칙 - 폰트: NanumGothic (본문 11pt, 제목 20pt Bold) - 색상: 제목 #191919, 강조 #4f46e5 - 로고: assets/logo.png (우측 하단) - 한 슬라이드에 내용 5줄 이내
STEP 1 · 보고서 초안
> 매출 분석 결과로 팀장 보고용 PPT 만들어줘. design.md 규칙 따라서
STEP 2 · 수정 지시
> 3페이지에 월별 추이 차트 추가하고, 표지에 로고 넣어줘
📄 마크다운 보고서
"보고서.md로 저장" → 빠른 텍스트 보고
📊 PPT 생성
"python-pptx로 .pptx 만들어줘" → 실제 파워포인트 파일
🎨 템플릿 유지
design.md 한 번 작성 → 모든 문서에 일관된 스타일
🎯 핵심: design.md에 우리 팀 규칙을 적어두면, 누가 만들어도 같은 스타일의 문서가 나옵니다.
